Colours in CSS

Hexidecimal eg. #000000

Colours can be set using a 6 digit hexi-decimal code, where each digit ranges from 0 (zero) - f (fifteen), where #000000 is black, and #ffffff is white.

The first digits represent the red channel, the second two, the green channel, and the third two the blue channel. This means that red is represented as #ff000000, green as #00ff00 and blue as #0000ff.

The number of colours available in each channel is 16 x 16 = 256, which means that in total there are 256 ^3 colour options in total, which is a whopping 16 777 216 colours to choose from.

3 digit Codes eg. #fff

To make things easier, we can also use 3 digit shorthand codes, with 1 digit representing each colour channel.

For example #bc1 is the same as the 6 digit code #bbcc11

Common Names

To make things easier again, some colours are given common names. While not all 16 777 216 colours have their own names, some common ones like #black, #white, #red do, as well as some more obscure ones like #greenyellow and #maroon. as well as hundreds of others.
