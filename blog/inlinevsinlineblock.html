CSS display: inline, inline-block, block

Introduction

To understand the differences in CSS display: formatting, between display:inline, display:inline-block, display:block.

These stle options are typically applied to the span tag.

display: inline

  • Top and bottom margins and & paddings are not respected.

  • This is the default display type if none specified.

  • Some default spans in a row.

display: inline-block

  • Allows a width and height of an element to be set.

  • Top and bottom margins and & paddings are respected.

  • More spans this time, display: inlineblock.

display: block

  • A line break happens after each element.

  • A block doesn't sit next to other elements.

  • Now, some spans, with display: block.
