What is the DOM?

The Document Object Model (DOM) is the data representation of objects that define the structure and layout of a website. Link

The DOM is useful to us because it allows us to change and modify the content, and the way that a website is displayed. Using JavaScript, we are able to update, insert, change and modify a websites look, feel and content in a programatic manner.

As an example, we could use JavaScript to select all "p" tags in the DOM, and make them dissapear, or change their colour.

There are a mirriad of JavaScript functions that allow us to access, update, and change objects in the DOM. As an example, "document.querySelectorAll("p")".

JavaScript is not the only language that can be used to access the DOM, for example, python could also be used to modify objects in the DOM, and hence Python could also be used to create and manage websites. Django is an example of this.

What is the difference between the DOM and HTML?

HTML is the standard markup language for creating websites. It defines the content and structure of a website.

The DOM

is the programming interface for web documents. It is not the same as HTML, however the DOM can be used to create HTML documents for displaying on the web.

The DOM contais nodes which represent HTML elements, attributes or text.

In summary, HTML is a markup language used to create web pages. DOM is a programming interface that represents a website, and can be changed programatically, which can then be displayed as HTML.

The DOM represents the 'current state' of a website, so as we change the elements and nodes in the DOM, the browser reflects those changes in HTML and the website is updated accordingly.
